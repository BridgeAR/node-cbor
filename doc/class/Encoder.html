Class: Encoder

Defined in: src/encoder.coffee
Inherits: stream.Readable

Overview

A Readable stream of CBOR bytes. Call write to get JSON objects translated into the stream.

Class Method Summary

  • . (void) encode(objs...) Encode one or more JavaScript objects, and return a Buffer containing the CBOR bytes.
