Class: Decoder

Defined in: src/decoder.coffee
Inherits: stream.Writable

Overview

Decode a CBOR byte stream into JavaScript objects. Either pipe another stream into an instance of this class, or pass a string, Buffer, or BufferStream into options.input, assign callbacks, then call start().
