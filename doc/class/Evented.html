Class: Evented

Defined in: src/evented.coffee
Inherits: stream.Writable

Overview

A SAX-style syntactic parser for CBOR. Either pipe another stream into an instance of this class, or pass a string, Buffer, or BufferStream into options.input, assign callbacks, then call start().

In the event callbacks, kind is one of the following strings:

  • 'value': an atomic value was detected
  • 'array-first': the first element of an array
  • 'array': an item after the first in an array
  • 'key-first': the first key in a map
  • 'key': a key other than the first in a map
  • 'stream-first': the first item in an indefinite encoding
  • 'stream': an item other than the first in an indefinite encoding
  • null: the end of a top-level CBOR item

Events:

value(val,tags,kind)

an atomic item (not a map or array) was detected

Parameters:

  • val ( any ) — the value
  • tags ( Array ) — an array of tags that preceded the value
  • kind ( String ) — see above

array-start(count,tags,kind)

the start of an array has been read.

Parameters:

  • count ( Integer ) — the number of items in the array. -1 if indefinite length.
  • tags ( Array ) — an array of tags that preceded the list.
  • kind ( String ) — see above

array-stop(count,tags,kind)

the end of an array has been reached.

Parameters:

  • count ( Integer ) — the actual number of items in the array.
  • tags ( Array ) — an array of tags that preceded the list.
  • kind ( String ) — see above

map-start(count,tags,kind)

the start of a map has been read.

Parameters:

  • count ( Integer ) — the number of pairs in the map. -1 if indefinite length.
  • tags ( Array ) — an array of tags that preceded the list
  • kind ( String ) — see above

map-stop(count,tags,kind)

the end of a map has been reached

Parameters:

  • count ( Integer ) — the actual number of pairs in the map.
  • tags ( Array ) — an array of tags that preceded the list
  • kind ( String ) — see above

stream-start(mt,tags,kind)

The start of a CBOR indefinite length bytestring or utf8-string. @param mt [] the major type for all of the items

Parameters:

  • tags ( Array ) — an array of tags that preceded the list
  • kind ( String ) — see above

stream-stop(count,mt,tags,kind)

we got to the end of a CBOR indefinite length bytestring or utf8-string. @param mt [] the major type for all of the items

Parameters:

  • count ( Integer ) — the number of constituent items
  • tags ( Array ) — an array of tags that preceded the list
  • kind ( String ) — see above

end()

the end of the input

error(er)

parse error such as invalid input

Parameters:
